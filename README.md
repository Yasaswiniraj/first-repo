# 🧮 Calculator

A modern desktop calculator built using **Python** and the **Tkinter** library. This project provides a clean graphical user interface for performing basic arithmetic operations with keyboard support and proper error handling.

---

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear button
- ⌨️ Keyboard support
- ⚠️ Handles invalid expressions gracefully
- 🎨 Clean and user-friendly interface

---

## 🛠️ Built With

- **Python 3**
- **Tkinter** (Python's built-in GUI library)

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your system

Check your Python version:

```bash
python --version
```

---

## ▶️ Run the Application

1. Clone the repository:

```bash
git clone https://github.com/Yasaswiniraj/first-repo.git
```

2. Navigate to the project folder:

```bash
cd first-repo
```

3. Run the application:

```bash
python src/main.py
```

> If your main file has a different name, replace `main.py` with the correct filename.
